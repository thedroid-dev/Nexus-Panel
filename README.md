# 🌌 NexusPanel

NexusPanel es un panel web ficticio diseñado para servidores Minecraft, con un estilo visual oscuro y moderno. Su objetivo es mostrar la capacidad de branding y diseño de interfaces.

---

## 🚀 Características
- Interfaz oscura con acentos puros y tipografía moderna  
- Secciones para administración de usuarios, rangos y economía  
- Diseño responsive para escritorio y móvil  
- Ejemplo de integración con APIs externas  

---

## 📦 Instalación
1. Clona este repositorio.  
2. Abre `index.html` en tu navegador.  
3. Personaliza los estilos en `styles.css`.

## 🚀 Explicación
- **index.html** → landing page con tu branding.  
- **styles.css** → tema oscuro, acentos puros, tipografía moderna.  
- **scripts.js** → lógica básica (ejemplo: abrir/cerrar menús, validaciones).  
- **assets/** → imágenes y recursos visuales.  
- **pages/** → subpáginas que simulan un panel real (usuarios, rangos, economía, etc.).  
- **README.md** → documentación clara y profesional.  
- **LICENSE** → para darle formalidad al repo.
